Arquitetura

Seam de repositório em três camadas: domain/repositories/* (interface) → data/mock/mock_* (lê assets/fixtures/*.json, simula latência) → data/providers.dart (binding Riverpod). Trocar mock por API real = mudar só o binding. Telas em Flame usam câmera cover (terreno preenche o frame) + _clamp() (pan travado).
